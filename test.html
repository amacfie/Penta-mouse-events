There's no DOM method for activating the :hover pseudo-class, according to this stackoverflow question, and Pentadactyl doesn't consider invisible elements for hints. Is there a Firefox XPCOM method for getting the active CSS statements and the elements they style? Manually grabbing and parsing the CSS files and style tags is an option. If finding the elements with :hover selectors in CSS is too hard, the brute-force method is hinting all elements. Perhaps sendMouseEventToWindow() could be used to simulate hovering on elements in a way that works for CSS.
